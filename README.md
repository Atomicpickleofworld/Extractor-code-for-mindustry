# IAExtractor — Inf.Adv Mod Dump Tool

A lightweight tool for Mindustry mod developers.  
Drag & drop a ZIP file → get a clean Markdown dump of all text-based files.

Perfect for:
- studying mod internals  
- building custom libraries  
- analyzing BulletTypes, effects, JSON content  
- merging multiple mods into a single reference file  

## Features
- English interface  
- Markdown output (.md)  
- File list at the top  
- Clean formatting with code blocks  
- Ignores binary files (PNG, OGG, WAV, ATLAS, etc.)  
- Output file name matches the ZIP name (ModName-Inf.Adv.md)  

## How to use
1. Run IAExtractor.exe
2. Drag & drop any Mindustry mod ZIP
3. Press Enter
4. Get a structured .md dump in the same folder
